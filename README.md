# How to build blog project with MERN

## Technology

-   Client: React
-   Server: NodeJS
-   DB: MongoDB

## Features

### For guest:

-   Sign up, Sign in
-   See detail post

### For user:

-   Sign out
-   Create, Edit post
-   Delete post
